# Cruise Finder Tool · Claude Code 插件

A powerful tool that assists users in finding the best cruises based on their preferences and requirements.

![published with craftwave](https://img.shields.io/badge/published%20with-craftwave-6366f1)

## 安装

在 Claude Code 中执行：

```
/plugin marketplace add OWNER/ola-tool-cruise-finder-claude-plugin
/plugin install ola-tool-cruise-finder@ola-tool-cruise-finder-marketplace
```

> 将 `OWNER` 替换为本仓库所属的 GitHub 账号/组织名。

安装后即可在 Claude Code 里使用技能 **Cruise Finder Tool**。本仓库由
[CruiseSkillBridge](https://cruiseskillbridge.com) 按 Claude Code 插件市场规范自动生成。
